## 功能
- [x] 登录注册(支持发送邮箱验证码)
- [x] 发帖(使用markdown编辑器，支持跨域上传本地图片)
- [x] 编辑自己的帖子(增删)
- [x] 首页(展示轮播图，每个板块最新的帖子，积分排行榜)
- [ ] 用户主页(资料设置，头像上传，查看自己的帖子与收藏的帖子)
- [ ] 帖子详情(html渲染，回复)
- [x] 搜索(展示搜索结果)
